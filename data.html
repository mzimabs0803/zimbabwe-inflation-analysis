Inflation Data & Forecasts (2010-2026)

Technical Methodology

The dataset was processed using the Pandas library in Python. Historical data (2010–2022) was pulled directly from the St. Louis Fed API. To provide a complete economic outlook, I manually integrated 2023–2024 actuals and 2025–2026 IMF stabilization targets into a unified data frame for visualization via Matplotlib.

Formal Data Sources
